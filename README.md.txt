# IBM Watsonx Travel Planner AI Agent

This project is an AI-powered Travel Planner Agent built using IBM Watsonx.ai Agent Lab.  
It helps users plan trips by suggesting destinations, itineraries, and budget-friendly travel tips.

## Features
- Generates day-wise travel itineraries
- Suggests tourist attractions and travel tips
- Provides budget-optimized plans
- Deployed as an AI service on IBM Cloud
- Tested with API endpoint and Python script

- **/screenshots** – Agent build, preview, and deployment screenshots
- **api_test.py** – Optional script to test deployed AI agent
